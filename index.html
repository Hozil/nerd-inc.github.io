Nerd!

Going Further

Nerd is an Open Source software organisation based in Bamenda, Cameroon. Nerd is a start-up that deals with machine learning, data mining, data analysis and knowledge extraction.

Open Source

We build the software for everyone

Nerd Inc is an open source organisation which aims at creating fun for the "nerd" :), by having fun with high abstract concepts (machine learning, UI hacking, lossless data compression to optimum scale).

Out-reach to community

We write code for everyone!

We are open source. But, we don't only concentrate on the developers but also on rural communities! Nerd aims to creates apps for the rural people in Cameroon who can't easily get their products to the market so that they get a global market to easily sell their products.
